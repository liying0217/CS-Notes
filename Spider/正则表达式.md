# Regular Expression

工具：https://regexr.com/

```
1234567890
abcdefghijklmnopqrstuvwxyz
ABCDEFGHIJKLMNOPQRSTUVWXYZ
+-*$
```



###### 匹配任意字符

- " . "匹配除断行外任意一个字符

![](img/r1.png)

- "\ ."代表转义，此时两个字符代表一个字符 

  ![](img/r2.png)

###### 匹配字母和数字

- "\w"用来匹配阿拉伯数字、英文大小写字母和下划线（w是小写的）

  ![](img/r3.png)

- “\W”匹配被“\w”排除在外的字符（\w的补集）

  ![](img/r4.png)

###### 匹配数字

- "\d"匹配阿拉伯数字

  ![](img/r5.png)

- "\d\\.\d"匹配小数

  ![](img/r6.png)

- "\D"匹配非阿拉伯数字（\d的补集）

  ![](img/r7.png)

###### 匹配空白字符

- "\s"匹配空白字符：空格、制表符、断行等

  ![](img/r8.png)

- "\S"匹配被"\s"排除在外的字符（\s的补集）

  ![](img/r9.png)

###### 字符集合

- [选择范围]匹配括号中的任意一个字符

  ![](img/r11.png)

- Unicode   万国码，用数字代表文字的一项标准

- [[\u4e00-\u9fa5]]匹配所有中文,共20901个，包括从“一”到“龥”

  ![](img/r10.png)

- [可以查看匹配的字符串](http://graphemica.com/)
